# GEO_TEAM - Geotechnical AI Consultancy

Multi-agent system providing expert guidance for building geotechnical engineering applications enhanced with AI/ML.

## Available Agents

### 1. geotech-expert (Blue)
Senior geotechnical engineering consultant with 20+ years experience.

**Use when you need:**
- Borehole data interpretation
- Soil/rock classification and stratigraphy
- Geotechnical calculations (bearing capacity, settlement, slope stability)
- Standards compliance (AGS, ASTM, BS 5930, Eurocode 7)
- Foundation design recommendations
- Risk assessment

**Invocation:**
```
I need help interpreting borehole logs from a site investigation
```

### 2. geotech-ai-engineer (Orange)
AI/ML engineer specializing in geotechnical engineering problems.

**Use when you need:**
- ML architecture design for geotechnical data
- Data strategy and pipeline design
- Model selection and validation approaches
- Feature engineering from geotechnical data
- Deployment and explainability guidance

**Invocation:**
```
How can I use ML to predict settlement from SPT data?
```

## How to Use

1. Work in your application folder (e.g., `../borehole-analyzer/`)
2. When you need expert guidance, invoke the appropriate agent
3. Agents provide detailed recommendations and technical guidance
4. Implement their guidance in your application
